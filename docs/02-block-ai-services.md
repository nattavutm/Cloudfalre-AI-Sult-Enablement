# Lab 2: Block AI Services

## Introduction
Learn how to use Gateway HTTP policies to block unauthorized AI applications.

## Steps
1. Go to **Gateway** -> **Firewall Policies** -> **HTTP**.
2. Create discovery policies for common AI categories.
3. Block specific applications like "ChatGPT", "Claude", and "Gemini".
4. Test the block from your browser.

## Verification
- Attempt to access `chatgpt.com`.
- You should see a Cloudflare block page or a connection refusal.
