# Lab 5: Approved AI Access

## Introduction
Provide secure, isolated access to approved AI tools.

## Steps
1. Create a "Browser Isolation" policy for approved apps (e.g., a corporate AI portal).
2. Set the action to "Isolate".
3. Configure restrictions like disabling copy/paste or file uploads.
4. Test the experience.

## Verification
- Access the isolated app.
- Look for the Cloudflare "Isolation" header or watermark.
