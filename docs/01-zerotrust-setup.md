# Lab 1: Zero Trust Setup

## Introduction
In this lab, you will initialize your Cloudflare Zero Trust environment.

## Steps
1. Log in to the [Cloudflare Dashboard](https://dash.cloudflare.com).
2. Navigate to **Zero Trust**.
3. Choose a team name.
4. Select the Free plan.
5. Configure your Identity Provider (OTP is the default).
6. Download and install the WARP client.
7. Enroll your device using the team name.

## Verification
- Open the WARP client and check if it says "Connected".
- Visit `https://help.teams.cloudflare.com` and verify "Connected to Cloudflare Gateway" is **Yes**.
